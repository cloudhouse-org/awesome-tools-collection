# 工具集锦

## Apps

### Clipboard

系统自带工具，Win + V 查看复制历史，选中即可重新复制，还可以辅助输入 Emoji 与特殊符号

![](.\assets\clipboard.png)

### 微信输入法

腾讯家的输入法，无广告，设置中可以配置多端同步剪贴板

### Everything + Everything Toolbar

Windows 自带搜索工具替代品。是最快速的文件搜索，资源占用极低，仅搜索文件和文件夹

Everything Toolbar 会将 Windows 系统下方 Dock 栏的搜索替换为 Everything，实现快速调用

![](.\assets\everything-toolbar.png)

使用场景：

- 寻找遗忘位置的文件
- 开发环境配置

### Powertoys

微软官方的开源多功能工具集合，内含一系列工具，增强 Windows 使用体验。

下面是其中几个我常用的小工具

- Always On Top：Ctrl + Win + T 固定当前窗口为最前
  - 使用场景：参考/对照某一窗口的内容
- Awake：定时强制电脑不熄屏
  - 使用场景：参考电脑内容用纸笔或者另一个设备干活；干活摸鱼电脑不熄屏
- Color Picker：Win + Shift + C 全局取色器（专业向）
  - 使用场景：UI、平面设计师
- File Locksmith：解决文件被占用导致的无法转移等等问题
  - ![](.\file-locksmith.png)

### Typora

Typora 是 Markdown 编辑器，快速实现简单排版，用于记电子笔记

使用场景：

- 快速记录整理
- 贴代码片段（可以高亮显示）

### PDF Arranger

用来调整 PDF 页面顺序的小软件

- [pdfarranger](https://gitee.com/mirrors_pdfarranger/pdfarranger)

### PDFMathTranslate

一个 PDF 翻译工具

- [PDFMathTranslate](https://github.com/Byaidu/PDFMathTranslate)

### KISS Translator

网页插件，双语对照翻译

![](.\assets\kiss.png)

- [kiss-translator](https://github.com/fishjar/kiss-translator)

### Pot

划词翻译工具 + OCR 工具。[Pot | 使用指南](https://pot-app.com/docs/)

### 沉浸式翻译

上面 KISS Translator 和 PDFMathTranslate 的整合网页插件，但是闭源且有收费项目

### Flow Launcher

针对倾向于键盘操作的用户的效率工具。Alt + 空格调用搜索框，可以实现快速打开软件，可以调用 Everything 搜索文件。Windows 实现键盘工作流

### GitHub Desktop

版本管理工具，直观的文件历史追溯。
